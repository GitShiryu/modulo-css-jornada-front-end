# CSS Journey: Front-End Fundamentals 🎨

This repository contains a comprehensive collection of studies and practical exercises developed during the **Jornada Front-End** course. It serves as a personal documentation of my mastery over CSS3 core concepts, from basic styling to advanced responsive layouts.

---

## 🚀 Project Overview

As an **Analysis and Systems Development** student, I believe that a strong grasp of fundamentals is key to building high-quality software. This repository organizes various modules of CSS learning, focusing on clean code, modular styling, and modern web standards.

## 📚 Topics Covered

The project is structured into several modules, each focusing on a specific CSS capability:

* **Selectors & Specificity:** Understanding how to target elements efficiently.
* **Box Model:** Mastering Margin, Padding, and Border relationships.
* **Layout & Visibility:** Deep dive into Display properties and element positioning.
* **Visual Effects:** Gradients, Shadows, Backgrounds, and Rounded Corners.
* **Typography:** Professional font handling and text styling.
* **Responsive Design:** Implementing flexible layouts for any screen size.

---

## 🛠 Tech Stack

* **HTML5:** Semantic structuring for educational modules.
* **CSS3:** Core styling engine, including Flexbox and advanced selectors.
* **Git:** Version control and repository management.

---

## 📂 Repository Structure

The files are organized numerically to follow the learning path:
- `001` to `005`: Core fundamentals (Selectors, Specificity, Inheritance).
- `006` to `013`: Visual styling (Colors, Units, Box Model, Backgrounds).
- `014` to `017`: Typography, Shadows, and **Responsiveness**.
